🌐 Personal Portfolio Website
📌 Overview

This repository contains the source code for my personal portfolio website, designed to showcase my projects, technical skills, and experience as a Computer Science Engineering student. The portfolio serves as a centralized platform for demonstrating my development capabilities and providing easy access to my work.

The website is built with a focus on performance, responsiveness, and clean UI/UX design, ensuring accessibility across devices and browsers.

🚀 Features
🧑‍💻 Fully responsive design (mobile-first approach)
⚡ Fast loading and optimized performance
🎨 Modern UI with clean layout
📂 Projects section with detailed descriptions
📄 Resume/CV integration
📞 Contact form / social links integration
🌐 Hosted using GitHub Pages
🛠️ Tech Stack
Frontend
HTML5 (Semantic structure)
CSS3 (Flexbox, Grid, Animations)
JavaScript (ES6+)
Tools & Platforms
Git & GitHub (Version Control)
GitHub Pages (Deployment)
VS Code (Development Environment)
